# Game Identity

## Concept
Top-down 2D pixel horror game. Player investigates haunted locations to identify the type of ghost using clues and equipment.

## Inspired By
Phasmophobia — but simpler, browser-based, solo.

## Core Loop
1. Enter a haunted location
2. Explore rooms, gather evidence
3. Identify the ghost type
4. Escape (or die trying)

## Tone
Tense, atmospheric, lo-fi pixel horror. Not gory — unsettling.

## MVP Features
- One map
- One ghost type
- 2–3 evidence tools (e.g. EMF reader, flashlight, journal)
- Win/lose condition

## Out of Scope (for now)
- Multiplayer
- Mobile
- Multiple ghost types (post-MVP)
